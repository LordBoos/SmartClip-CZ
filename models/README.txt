Czech Vosk Model Installation Instructions
==========================================

1. Download the Czech model from:
   https://alphacephei.com/vosk/models

2. Look for: vosk-model-small-cs-0.4-rhasspy.zip

3. Extract the ZIP file here so you have:
   models/vosk-model-small-cs-0.4-rhasspy/

4. The final structure should be:
   models/
   └── vosk-model-small-cs-0.4-rhasspy/
       ├── am/
       ├── graph/
       ├── ivector/
       └── conf/

5. Restart OBS and reload the SmartClip CZ script.

Note: The model is about 40MB compressed, 120MB extracted.
